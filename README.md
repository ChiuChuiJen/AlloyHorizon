# Mecha RPG (Web) — V0.0.1

這是一個可直接用 GitHub Pages 掛載執行的靜態網頁遊戲骨架（機甲RPG）。

## 功能
- 打怪、升級、掉寶
- 裝備欄位：頭、身體、左手、右手、腳、配件1、配件2、配件3
- 狀態：HP / MP / 能量 / 同步率(EXE)
- 儲存/讀檔（localStorage）
- 匯入/匯出：JSON / Base64
- 版本更新內容：顯示近三筆，其餘可在彈窗查看

## GitHub Pages
1. 將本資料夾內容推到 GitHub Repo 根目錄
2. Settings → Pages → Branch: main / Folder: /(root)
3. 打開 Pages 提供的網址即可遊玩
